## Important note!
The quotes.txt file must be created manually in advance!


## This bot has the following features:
The main loop checks if the message belongs to a user on the monitoring list.</br>
If the user is on the monitoring list, the bot checks every word in their message.</br>
If a swear word is found, the bot deletes the user's message.</br>
if not, it does nothing.


## add_user
Adds a user to the monitoring list in the `users.txt` file.


## quote
Sends a random quote from the `quotes.txt` file.

## remove_user
Removes a user from the monitoring list in the `users.txt` file.

## uptime
Sends the bot's uptime.

## teach
Adds a new question-answer pair.

## ask
Asks the bot a question.
